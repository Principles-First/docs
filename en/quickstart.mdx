# Quick Start Guide

Get your AI assistant up and running in just a few steps.

<Info>
    You'll need your **Application ID** from the [Moss
    Dashboard](https://dashboard.viamoss.ai) and a JWT signing key for authentication.
</Info>

## 1. Install the SDK

```bash
npm install @viamoss/moss-sdk
```

## 2. Add Moss to Your App

Add `AgentProvider` and `AssistantButton` to your application. The provider does **not** need to wrap your entire app â€” just the Moss components.

```tsx App.tsx
import { AgentProvider, AssistantButton } from '@viamoss/moss-sdk';

function App() {
  return (
    <>
      <YourApp />
      <AgentProvider config={{
        apiUrl: 'https://clippy-api.viamoss.ai',
        applicationId: 'YOUR_APP_ID',
        userId: currentUser.id,
        getJwt: () => fetchMossToken(),
      }}>
        <AssistantButton />
      </AgentProvider>
    </>
  );
}
```

<Tip>
    `AgentProvider` only needs to wrap the Moss UI components. It observes the full page DOM regardless of where it sits in the React tree.
</Tip>

## 3. Set Up Authentication

Create a backend endpoint to generate JWT tokens for your users. See the [Authentication Guide](/en/sdk/authentication) for complete setup instructions.

```typescript api/moss-token.ts
// Example Next.js API route
import jwt from 'jsonwebtoken';

export async function GET(request: Request) {
  const user = await getCurrentUser(request);

  const token = jwt.sign(
    { sub: user.id, app: 'YOUR_APP_ID' },
    process.env.MOSS_JWT_SECRET,
    { expiresIn: '1h' }
  );

  return Response.json({ token });
}
```
