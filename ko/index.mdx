## Moss에 오신 것을 환영합니다

Moss는 웹 애플리케이션에 직접 임베드되는 AI 기반 어시스턴트 플랫폼입니다. 상황에 맞는 도움말, 단계별 가이드, 지능형 지원을 제공하여 사용자가 원활하게 작업을 탐색하고 수행할 수 있도록 도와줍니다.

### 주요 기능

- **상황 인식** - Moss는 사용자가 보는 것을 보고, 현재 화면과 작업에 따라 관련 지원을 제공합니다
- **단계별 가이드** - 복잡한 작업을 시각적 하이라이팅과 함께 관리 가능한 단계로 분해합니다
- **지식 기반 통합** - 정확하고 최신 답변을 위해 문서를 연결합니다
- **세션 리플레이** - 사용자 상호작용을 검토하여 제품과 지원을 개선합니다
