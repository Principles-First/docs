## 핵심 기능

| 설정 | 타입 | 기본값 | 위치 |
|---|---|---|---|
| `enableVision` | `boolean` | `true` | Dashboard + SDK |
| `enableSessionRecording` | `boolean` | `true` | Dashboard + SDK |
| `enableScreenHistory` | `boolean` | `true` | Dashboard + SDK |

## 세션

| 설정 | 타입 | 기본값 | 위치 |
|---|---|---|---|
| `sessionConfig.inactivityTimeout` | `number` (분) | `30` | Dashboard + SDK |
| `sessionConfig.maxSessionAge` | `number` (시간) | `2` | Dashboard + SDK |
| `sessionConfig.showNotification` | `boolean` | `true` | Dashboard + SDK |

## 플래닝

| 설정 | 타입 | 기본값 | 위치 |
