# 빠른 시작 가이드

몇 단계만으로 AI 어시스턴트를 실행하세요.

<Info>
    [Moss 대시보드](https://dashboard.viamoss.ai)에서 **애플리케이션 ID**와
    인증을 위한 JWT 서명 키가 필요합니다.
</Info>

## 1. SDK 설치

```bash
npm install @viamoss/moss-sdk
```

## 2. 앱에 Moss 추가

애플리케이션에 `AgentProvider`와 `AssistantButton`을 추가하세요. provider는 전체 앱을 감쌀 필요가 **없습니다** — Moss 컴포넌트만 감싸면 됩니다.

```tsx App.tsx
import { AgentProvider, AssistantButton } from '@viamoss/moss-sdk';

function App() {
  return (
    <>
      <YourApp />
      <AgentProvider config={{
        apiUrl: 'https://clippy-api.viamoss.ai',
        applicationId: 'YOUR_APP_ID',
        userId: currentUser.id,
        getJwt: () => fetchMossToken(),
      }}>
        <AssistantButton />
      </AgentProvider>
    </>
  );
}
```

<Tip>
    `AgentProvider`는 Moss UI 컴포넌트만 감싸면 됩니다. React 트리에서 어디에
    위치하든 전체 페이지 DOM을 관찰합니다.
</Tip>

## 3. 인증 설정

사용자를 위한 JWT 토큰을 생성하는 백엔드 엔드포인트를 만드세요. 전체 설정 방법은 [인증 가이드](/ko/sdk/authentication)를 참조하세요.

```typescript api/moss-token.ts
// Next.js API 라우트 예시
import jwt from 'jsonwebtoken';

export async function GET(request: Request) {
  const user = await getCurrentUser(request);

  const token = jwt.sign(
    { sub: user.id, app: 'YOUR_APP_ID' },
    process.env.MOSS_JWT_SECRET,
    { expiresIn: '1h' }
  );

  return Response.json({ token });
}
```
